VPython and Glowscript

VPython is Python with added 3D graphic modules, and some other function shortcuts that just makes it easier to use than pure Python. It is specially convenient in physics education, because of the ease to visualize motion, and use vectors in the codes.

This is a link with documentation, YouTube videos and introduction to VPython.

There are two ways to use VPython:

1) by installing a complete Python package in your computer (for more experienced users):

If you choose to install Anaconda in your computer, get it here: https://www.anaconda.com/download/

2) or installing nothing, and instead running the VPython codes in an internet browser.

I find that for most classroom applications, running codes in a browser is the best option. You can send a link with the codes to students, who can then make changes, visualize the results in the browser, and re-submit a link to you in an easy manner. Here are a few environments that run VPython in a browser (note: always avoid Internet Explorer because of frequent incompatibilities; instead, use Safari/Firefox/Chrome/etc):

http://www.glowscript.org

https://trinket.io

What is Glowscript? Glowscript is simply an environment that reads VPython and runs it in a browser for you. So, the first line of every code you or your students write in VPython (while in the browser enviroment) will be "Glowscript 2.6 VPython". The version is always changing, so 2.6 may become 2.7 etc.

I acknowledge several AAPT workshops that I have attended over the years, as the source for inspiration, initial instruction and resources, that helped me find the way to integrate VPython codes in my classroom.

Some codes for the classroom

PHY130 - mechanics - 1st semesters of a 3-semester sequence of calculus-based physics

I give small coding challenges to students using trinket.io. Explanations are given as commented lines (shown in green) at the beginning of the codes. The challenges required small modifications in the original codes. The focus is in the physics, not in the VPython language.

Challenge #1: This is the first coding challenge introduced to my PHY130 students at the start of the semester. Students start from this minimal working example, and add/modify a few lines to achieve what is asked in the commented lines. First, students need to open a free account in trinket.io, to be able to save and modify the codes in each of their personal accounts. Trinket.io offers the option to share the link of a code saved in your account. Therefore, once the work is done, students only need to send the new link.

Click on the ▶ Run symbol to see the result of this starting code.

Below is the desired final result of challenge #1. I am not displaying the final code, only the vizualization of the final result, so that my students can still work with this exercise in the future.
